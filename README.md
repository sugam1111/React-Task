# User Management Portal

## Project Description

This project is a User Management Portal built using React and TypeScript.  
It allows authenticated users to log in and manage a list of users.

Main features include:
- User login using API authentication
- Viewing a list of users with pagination
- Searching users by name, email, or username
- Filtering users by gender and age range
- Sorting users by name, age, email, and username
- Adding todos for individual users
- Viewing todos assigned to a user
- Saving login state using localStorage

The project uses DummyJSON as a mock backend API and focuses on clean UI, reusable components, and proper state handling.
